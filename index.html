🚀 欢迎来到 我的小项目！

这里是Docker镜像仓库，致力于为您提供高效、便捷的镜像服务。

为了让您能够顺利地拉取和使用本仓库中的镜像，请参考以下操作指南：

  • 方法一：直接使用 `docker pull` 命令拉取镜像
    在某些配置下，您可以直接使用 `docker pull` 命令来拉取镜像。 
    docker pull liuasd-dcor.edgeone.app/your-image-name:tag
  • 方法二：修改Docker配置

    操作步骤（仅适用于Linux命令行环境）：

    1. 编辑或创建 `daemon.json` 文件：
      使用您喜欢的文本编辑器打开或创建Docker守护进程的配置文件 `/etc/docker/daemon.json`。例如： 
      sudo vim /etc/docker/daemon.json
    2. 添加 `insecure-registries` 配置：
      如果文件为空或不存在，请添加以下内容： 
      {
  "insecure-registries": [
    "liuasd-dcor.edgeone.app"
  ]
}
      如果文件中已有其他配置，请确保将其添加到现有JSON对象的顶部或底部，并用逗号分隔（如果不是最后一个条目）： 
      {
  "other-config": "value",
  "insecure-registries": [
    "liuasd-dcor.edgeone.app"
  ]
}
    3. 重启Docker守护进程：
      保存 `daemon.json` 文件后，您需要重启Docker守护进程以使更改生效。 
      sudo systemctl daemon-reload
sudo systemctl restart docker
    4. 验证配置并拉取镜像：
      重启后，您现在应该能够成功从 `liuasd-dcor.edgeone.app` 拉取镜像了： 
      docker pull liuasd-dcor.edgeone.app/your-image-name:tag
  • 遇到问题？
    如果在配置或拉取过程中遇到任何问题，请检查您的网络连接、`daemon.json` 文件语法是否正确，并确保Docker守护进程已成功重启。

我们致力于提供一个稳定、便捷的Docker镜像服务。祝您使用愉快！
